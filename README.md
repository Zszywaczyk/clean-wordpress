# clean-wordpress

A clean, professional WordPress starter theme built for real client projects. Gulp-powered build system with Bootstrap, SCSS compilation, JS bundling, live reload, and production-ready minification.

Built from experience building multiple commercial WordPress sites. No bloat — only what gets used on real projects.

## Tech Stack

| Layer | Tools |
|---|---|
| **CSS** | SCSS → Bootstrap 4 (custom build), cleanCSS (level 2), media query merging |
| **JS** | Vanilla JS → UglifyJS, vanilla-lazyload (included in bundle) |
| **Build** | Gulp 4, gulp-include (JS bundling), sourcemaps |
| **Dev** | browser-sync (proxy), watch tasks for PHP/SCSS/JS |
| **Production** | `gulp build --env=production` — minifies CSS+JS, merges media queries, removes sourcemaps |

## Quick Start

### 1. Install WordPress via WP-CLI

```bash
wp core download --locale=pl_PL

wp config create --dbname=yourproject --dbuser=root --dbpass= --locale=pl_PL --dbhost=localhost

wp core install --url=yoursite.local --title="Your Site Title" --admin_name=admin --admin_email=your@email.com --admin_password=your_password
wp option set blog_public 0
```

### 2. Install dependencies

```bash
npm install
```

### 3. Set browser-sync proxy

In `gulpfile.js`, update the proxy to match your local URL:

```js
browserSync.init({
    proxy: "http://yoursite.local",
    notify: false
});
```

### 4. Run

```bash
# Development (watch + browser-sync + sourcemaps)
gulp

# Production build (minified CSS+JS, no sourcemaps)
gulp build --env=production
```

## Notes

- `partials-example/` contains reference implementations for each partial — copy to `partials/` and customize
- Bootstrap is compiled from source via SCSS — only import what you need in `src/sass/main.scss`
- vanilla-lazyload is bundled into `main.js` via `gulp-include`
